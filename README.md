# How to run and edit
### Terminal 1
```
cd .\backend
npm i
npm run watch
```
Backend started at [`http://localhost:4443`](http://localhost:4443)
### Terminal 2
```
cd .\frontend
npm i
ng serve
```
Frontend started at [`http://localhost:4200`](http://localhost:4200)
